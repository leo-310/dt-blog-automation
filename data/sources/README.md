# Source Library

This folder is the agent's monthly input stack.

Add notes here before generating a new wave of content. The generator automatically reads all markdown files in this folder and feeds them to the prompt.

Recommended folders:

- `brand/`: product specs, materials, certifications, wash instructions, founder reasoning
- `research/`: approved study notes from PubMed, AAD, journals
- `customer_language/`: reviews, emails, DMs, Reddit phrasing
- `competitor_gaps/`: queries competitors rank for, content gaps, titles to beat

Keep each note short and factual. The agent works best when you summarize the useful parts instead of pasting huge raw dumps.
